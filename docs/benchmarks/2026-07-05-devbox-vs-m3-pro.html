July 5, 2026 benchmark summary

Devbox wins command loops. Strict-local storage cuts the IO gap.

This visual summary accompanies the Markdown result doc. It compares the KubeVirt devbox to the M3 Pro Mac and shows what changed after recreating the devbox VM with the `longhorn-devbox-local` storage class.
